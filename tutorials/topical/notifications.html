通知¶

注解

有关与任务交互函数的完整列表，请见 C API 和 C++ API。

任务通知是 PROS 3 中强大的一个新功能，它允许面向任务的同步。每个任务都有一个 32 位的通知值, 可以跟据自己的通知（等待它变为非 0）来进行阻塞，其他任务可以更新这个通知值。 与传统的基于信号量的同步机制相比，任务通知具有广泛的应用范围，以及易于使用以及在内存和速度上显著的优势。

任务通知的最简单应用不关心任务通知值：
