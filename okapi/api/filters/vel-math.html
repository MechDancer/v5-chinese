okapi::VelMath¶

Velocity math helper. Calculates filtered velocity. Filters using a 2-tap averaging filter by default, unless a different filter is given. If you are trying to make an instance of this class, you should most likely be using the VelMathFactory instead of a constructor from this class.

Constructor(s)¶

Throws a std::invalid_argument exception if iticksPerRev is zero.

Prototype

Parameters  
iticksPerRev The number of ticks per revolution (or whatever units you are using).
ifilter The filter used for filtering the calculated velocity.
isampleTime The minimum time between velocity measurements.
iloopDtTimer The timer used for calculating loop dt’s.
